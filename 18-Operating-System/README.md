# Operating System

## Overview
An Operating System (OS) is system software that manages computer hardware and software resources, providing common services for computer programs. It acts as an intermediary between user applications and the underlying hardware.

## OS Components

`
Operating System
├── Kernel (core)
│   ├── Process Scheduler
│   ├── Memory Manager
│   ├── File System
│   ├── Device Drivers
│   ├── Network Stack
│   └── IPC (Inter-Process Communication)
├── System Libraries (DLLs, shared objects)
└── System Utilities (shell, file tools, config)
`

## Types of Operating Systems

| Type | Description | Examples |
|------|-------------|----------|
| Batch | Jobs batched together, no user interaction | IBM OS/360 |
| Time-Sharing | CPU time shared among users | Unix, Linux |
| Real-Time | Deterministic response within deadlines | RTOS, VxWorks |
| Distributed | Manages a group of independent computers | Plan 9, Amoeba |
| Embedded | Dedicated function within a device | FreeRTOS, Zephyr |
| Mobile | Designed for smartphones/tablets | Android, iOS |
| Virtualization | Hosts multiple guest OSes | Hyper-V, VMware ESXi |
