# JavaScript Programming Language

## Overview

JavaScript is a high-level, interpreted, dynamically-typed language originally created for web browsers. With Node.js, it expanded to server-side, mobile (React Native), and desktop (Electron) development. Modern JavaScript (ES6+) includes classes, modules, async/await, and functional programming features.

## Key Features

- **Multi-Paradigm**: Object-oriented (prototypal + classes), functional, and event-driven
- **First-Class Functions**: Functions as values — pass, return, store them
- **Asynchronous**: Native `async`/`await`, Promises, event loop
- **Prototypal Inheritance**: Objects delegate to other objects (not class-based copying)
- **Dynamic Typing**: Types checked at runtime, flexible but requires discipline
- **Massive Ecosystem**: npm has 1M+ packages for every use case

## Directory Structure

- `Basics/` — Syntax, closures, async/await, modules, prototypal inheritance
- `Algorithms/` — Algorithm implementations in JavaScript
- `Data-Structures/` — Data structure implementations in JavaScript
- `OOP/` — Prototypal patterns, ES6 classes, factory functions, composition

## Recommended Learning Path

1. Start with the [Introduction](javascript-introduction.md) for setup and motivation
2. Work through `Basics/` for core syntax and async patterns
3. Study `Data-Structures/` for arrays, objects, and custom structures
4. Practice with `Algorithms/` for sorting, searching, and string problems
5. Explore `OOP/` for prototypes, classes, and composition patterns

## JavaScript vs Other Languages

| Feature | JavaScript | TypeScript | Java | Python |
|---------|-----------|------------|------|--------|
| Typing | Dynamic | Static (compiles to JS) | Static | Dynamic |
| Inheritance | Prototypal | Prototypal | Class-based | Class-based |
| Async | Promises + `async/await` | Same | Threads + Futures | `asyncio` |
| Compilation | Interpreted (JIT) | Compiled to JS | Bytecode (JVM) | Interpreted |
| Module System | ES Modules | ES Modules | Packages | Imports |
| Use Case | Web (browser + server) | Web + large apps | Enterprise, Android | Scripting, data, web |
