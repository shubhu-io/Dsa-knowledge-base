# JavaScript Introduction

## Why Learn JavaScript?
JavaScript is the language of the web, used for frontend, backend, and mobile development.

## Key Features
- **Browser Native**: Runs in all browsers
- **Event-Driven**: Asynchronous programming
- **Functional & OOP**: Multiple paradigms
- **Huge Ecosystem**: npm has 1M+ packages
- **Full-Stack**: Frontend + Backend (Node.js)

## Getting Started

### Installation
1. Install Node.js
2. Or use browser console

### First Program
```javascript
console.log("Hello, World!");
```

## Basic Syntax

### Variables
```javascript
let name = "Alice";      // Mutable
const age = 30;          // Immutable
```

### Control Flow
```javascript
if (age >= 18) {
    console.log("Adult");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

## Best Practices

1. Use const by default
2. Use let when needed
3. Avoid var
4. Use arrow functions
5. Use async/await
