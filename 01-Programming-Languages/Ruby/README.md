# Ruby Programming Language

## Overview

Ruby is a dynamic, object-oriented programming language created by Yukihiro "Matz" Matsumoto in 1995. Designed for programmer happiness and productivity, Ruby emphasizes convention over configuration and provides an elegant syntax that reads like natural language. It is best known as the language behind Ruby on Rails, one of the most influential web frameworks.

## Key Features

- **Everything is an Object**: Even primitives like integers and booleans are objects
- **Elegant Syntax**: Minimal boilerplate, natural language-like constructs
- **Powerful Metaprogramming**: Dynamic method definition, `method_missing`, open classes
- **Blocks and Procs**: First-class closures that enable expressive DSLs
- **Rich Ecosystem**: Gems package manager, Ruby on Rails, Sinatra, and more

## Directory Structure

- `Basics/` — Variables, data types, blocks, procs, lambdas, metaprogramming, DSLs
- `Algorithms/` — Algorithm implementations in Ruby including string algorithms
- `Data-Structures/` — Data structure implementations in Ruby
- `OOP/` — Classes, modules, mixins, metaprogramming, `method_missing`, open classes

## Recommended Learning Path

1. Start with the [Introduction](ruby-introduction.md) for setup and motivation
2. Work through `Basics/` for core Ruby syntax and its object-centric philosophy
3. Study `Data-Structures/` for arrays, hashes, linked lists, trees
4. Practice with `Algorithms/` for sorting, searching, and string problems
5. Explore `OOP/` for modules, mixins, metaprogramming, and design patterns

## Ruby vs Other Languages

| Feature | Ruby | Python | JavaScript | PHP |
|---------|------|--------|------------|-----|
| Philosophy | Programmer happiness | Readability | Versatility | Web-first |
| Type System | Dynamic | Dynamic | Dynamic | Dynamic + optional static |
| Web Framework | Rails, Sinatra | Django, Flask | Express, Next.js | Laravel, Symfony |
| Package Manager | gem | pip | npm | composer |
| Metaprogramming | Extensive (open classes, `method_missing`) | Moderate (`getattr`) | Moderate (proxies) | Moderate (magic methods) |
| Concurrency | GIL (CRuby), Ractor (3.0+) | GIL | Event loop | Fibers, coroutines |
