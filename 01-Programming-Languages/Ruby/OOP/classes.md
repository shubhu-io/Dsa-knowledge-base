# Ruby Classes Example

## Shape Hierarchy

This file demonstrates a shape hierarchy using Ruby's OOP features:

- **Module mixins**: `Drawable`, `Calculable` for code reuse
- **Inheritance**: `Circle` and `Rectangle` extend `Shape`
- **Metaprogramming**: Dynamic field definition

```ruby
# Run: ruby OOP/classes.rb
# Output shows all shapes with areas and perimeters
```
