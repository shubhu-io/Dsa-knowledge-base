# PHP Classes Example

## Shape Hierarchy

This file demonstrates a shape hierarchy using PHP's OOP features:

- **Abstract class**: `Shape` provides common behavior
- **Interface**: `Drawable`, `Resizable`, `Comparable`
- **Traits**: `Timestampable`, `Printable` for code reuse
- **Concrete classes**: `Circle`, `Rectangle`, `Square`, `Triangle`

```php
// Run: php OOP/classes.php
// Output shows all shapes with areas, perimeters, and descriptions
```
