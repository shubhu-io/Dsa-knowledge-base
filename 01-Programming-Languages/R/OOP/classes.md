# R Classes Example

## Shape Hierarchy

This file demonstrates shape hierarchies using R's different OOP systems:

- **S3 classes**: Informal OOP with function dispatch
- **R6 classes**: Reference-based OOP with encapsulation

```r
# Run: Rscript OOP/classes.R
# Output shows shapes using both S3 and R6 approaches
```
