# R Programming Language

## Overview

R is a programming language and environment designed for statistical computing and data analysis. Created by Ross Ihaka and Robert Gentleman in 1993 at the University of Auckland, R has become the dominant language in academia and industry for data science, bioinformatics, and statistical modeling. Its power comes from a rich ecosystem of packages and unmatched statistical capabilities.

## Key Features

- **Statistical Powerhouse**: Thousands of packages for every statistical method imaginable
- **Data Visualization**: ggplot2 is widely considered the best data visualization library
- **Vectorized Operations**: Operates on entire data structures without explicit loops
- **Multiple OOP Systems**: S3, S4, R6, and Reference Classes for different needs
- **Reproducible Research**: R Markdown and Quarto for literate programming

## Directory Structure

- `Basics/` — Vectors, matrices, data frames, lists, factors, apply family, tidyverse
- `Algorithms/` — Algorithm implementations in R including string algorithms
- `Data-Structures/` — Data structure implementations in R
- `OOP/` — S3, S4, R6 class systems, when to use each approach

## Recommended Learning Path

1. Start with the [Introduction](r-introduction.md) for setup and motivation
2. Work through `Basics/` for core R syntax, data types, and the tidyverse
3. Study `Data-Structures/` for vectors, lists, data frames, and tibbles
4. Practice with `Algorithms/` for sorting, searching, and string problems
5. Explore `OOP/` for S3, S4, and R6 class systems

## R vs Other Languages

| Feature | R | Python | Julia | SAS |
|---------|---|--------|-------|-----|
| Primary Use | Statistics/data science | General purpose | Scientific computing | Statistical analysis |
| Type System | Dynamic | Dynamic | Dynamic + optional static | Macro-based |
| Data Viz | ggplot2, lattice | matplotlib, seaborn | Plots.jl | Built-in charts |
| OOP | S3, S4, R6 | Classes, dataclasses | Multiple dispatch | Not applicable |
| Package Manager | CRAN, Bioconductor | PyPI, conda | General Registry | N/A |
| Learning Curve | Steep (unique syntax) | Gentle | Moderate | Steep (proprietary) |
