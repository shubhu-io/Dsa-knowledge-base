# Kotlin

## Overview

Kotlin is a modern, statically typed programming language that runs on the JVM, compiles to JavaScript, and can be compiled to native code. Developed by JetBrains, it is fully interoperable with Java and has become the preferred language for Android development since Google's official adoption in 2017.

## Key Features

- **Null Safety**: The type system distinguishes between nullable and non-nullable types, eliminating null pointer exceptions at compile time.
- **Coroutines**: First-class support for asynchronous programming with lightweight threads.
- **Data Classes**: Concise syntax for classes that primarily hold data.
- **Extension Functions**: Add functionality to existing classes without inheritance.
- **Smart Casts**: The compiler tracks type information and inserts casts automatically after type checks.
- **Coroutines & Flow**: Structured concurrency for asynchronous and reactive programming.
- **Interoperability**: Seamless two-way interoperability with Java.

## JVM Interop

Kotlin is designed to be fully interoperable with the Java ecosystem:

- Kotlin code compiles to JVM bytecode and can be called from Java.
- Java code can be called from Kotlin with no extra effort.
- Kotlin collections map to Java collections under the hood.
- Kotlin uses Java's standard library (`kotlin-stdlib` extends `java-stdlib`).
- Annotation processing (e.g., Room, Dagger) works in Kotlin.

```kotlin
// Calling Java from Kotlin
val list = ArrayList<String>()
list.add("Hello")
list.forEach { println(it) }

// Calling Kotlin from Java (generated getters/setters work automatically)
val user = User("Alice")
println(user.getName()) // Generated by Kotlin
```
