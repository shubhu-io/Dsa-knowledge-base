# Scala Classes Example

## Shape Hierarchy

This file demonstrates a shape hierarchy using Scala's OOP features:

- **Traits**: `Drawable`, `Resizable`, `PrettyPrint`
- **Case classes**: `Circle`, `Rectangle` with pattern matching
- **Companion objects**: Factory methods
- **Pattern matching**: Exhaustive shape handling

```scala
// Run: scala OOP/classes.scala
// Output shows shapes with areas and descriptions
```
