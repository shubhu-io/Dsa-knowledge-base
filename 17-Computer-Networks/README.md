# Computer Networks

## Overview
Computer networking is the practice of connecting computers and devices to share resources and communicate. Networks form the backbone of modern internet, enabling everything from web browsing and email to cloud computing and IoT.

## Key Concepts at a Glance

`
Computer Networks
├── Network Types
│   ├── PAN (Personal Area Network) — ~10m
│   ├── LAN (Local Area Network) — ~1km
│   ├── MAN (Metropolitan Area Network) — ~50km
│   └── WAN (Wide Area Network) — global
├── Topologies
│   ├── Bus, Star, Ring, Mesh, Tree, Hybrid
├── Models
│   ├── OSI Model (7 layers)
│   └── TCP/IP Model (4/5 layers)
├── Protocols
│   ├── Application: HTTP, DNS, SMTP, FTP
│   ├── Transport: TCP, UDP
│   ├── Network: IP, ICMP, ARP
│   └── Link: Ethernet, Wi-Fi, PPP
└── Devices
    ├── Hub (L1), Switch (L2), Router (L3)
    ├── Bridge (L2), Gateway (L4-7)
    └── Modem, Repeater, Access Point
`
