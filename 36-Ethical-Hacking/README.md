# Ethical Hacking

## Overview

Ethical Hacking involves authorized attempts to gain unauthorized access to systems, applications, or data by using the same strategies and tools as malicious hackers, but in a legal and legitimate manner to identify vulnerabilities.

## What is Ethical Hacking?

Ethical hackers (also known as white-hat hackers) are cybersecurity professionals who:

- Have explicit permission to test system security
- Follow established rules of engagement
- Report vulnerabilities to system owners
- Help improve security posture
- Follow legal and ethical guidelines

## Legal Framework

### Authorized Testing Only

```
⚠️ IMPORTANT: Only test systems you own or have 
written authorization to test.

Legal documents needed:
• Rules of Engagement (ROE)
• Statement of Work (SOW)
• Permission letter
• Non-disclosure agreement (NDA)
```
