# Searching Algorithms

## Overview

Searching algorithms find the position of a target element within a data structure or collection.

## Common Searching Algorithms

| Algorithm | Time (Best) | Time (Avg) | Time (Worst) | Space | Requirement |
|-----------|-------------|------------|--------------|-------|-------------|
| Linear Search | O(1) | O(n) | O(n) | O(1) | None |
| Binary Search | O(1) | O(log n) | O(log n) | O(1) | Sorted |
| Ternary Search | O(1) | O(log₃ n) | O(log₃ n) | O(1) | Sorted |

## When to Use

- **Unsorted data**: Linear Search
- **Sorted data**: Binary Search
- **Peak finding**: Binary Search variant
