# Dynamic Programming

This directory contains learning materials for dynamic programming, a powerful algorithmic technique for solving complex problems by breaking them down into simpler overlapping subproblems.

## Contents

- [dp-overview.md](dp-overview.md) - Overview of dynamic programming concepts and when to use them
- [dp-problems.md](dp-problems.md) - Collection of practice problems categorized by difficulty (Easy, Medium, Hard) with solutions
- [dp-cheatsheet.md](dp-cheatsheet.md) - Quick reference guide with key patterns, templates, and optimization strategies
- [README.md](README.md) - This file

## Topics Covered

### Core Principles
- Overlapping subproblems and optimal substructure
- Memoization vs tabulation approaches
- State definition and transition formulation

### Common DP Patterns
- 1D DP (linear sequences)
- 2D DP (grids and matrices)
- Knapsack variations
- String DP (edit distance, LCS)
- Interval DP
- Tree DP
- Bitmask DP

### Optimization Techniques
- Space complexity reduction
- Time complexity improvements
- Memoization strategies
- Problem-specific optimizations

### Problem Categories
- Sequence/array problems (LIS, maximum subarray)
- String problems (edit distance, word break)
- Grid problems (unique paths, minimum path sum)
- Knapsack problems (0/1, unbounded, coin change)
- Tree problems (maximum path sum, diameter)
- Advanced problems (TSP, DP with optimization)

## Learning Path

1. **Beginner**: Start with the overview to understand core concepts
2. **Intermediate**: Practice with easy and medium problems, learn basic patterns
3. **Advanced**: Tackle hard problems, study optimization techniques
4. **Mastery**: Apply DP to novel problems, analyze and optimize solutions

## Related Topics
After mastering dynamic programming, consider exploring:
- Greedy algorithms (when DP can be optimized)
- Divide and conquer (alternative approach)
- Graph algorithms (often combined with DP)
- Advanced data structures (for efficient DP implementations)

## Prerequisites
- Basic programming proficiency (recursion, loops, conditionals)
- Understanding of time/space complexity analysis
- Familiarity with fundamental data structures (arrays, lists, maps)
- Basic mathematical reasoning

## Applications
Dynamic programming is widely used in:
- Bioinformatics (DNA sequence alignment)
- Operations research (resource allocation)
- Computer science (compiler optimization, graphics)
- Economics and finance (investment strategies)
- Engineering (control systems, signal processing)
