# 07 - LeetCode

## Overview

LeetCode is the premier platform for technical interview preparation and algorithm practice. With over 2,700+ problems covering data structures, algorithms, and system design, it is the go-to resource for software engineers preparing for coding interviews at top tech companies.

## Why LeetCode?

- **Industry Standard**: Most FAANG and top tech companies use LeetCode-style questions in interviews
- **Structured Learning**: Problems organized by difficulty, topic, and company
- **Contest System**: Weekly and bi-weekly contests to test your skills under pressure
- **Company-Specific Tags**: See which companies have asked specific problems
- **Discussion Forums**: Learn from community solutions and explanations

## Recommended Progression

```
Phase 1: Fundamentals (Weeks 1-4)
  └── 50 Easy problems across all topics

Phase 2: Core Skills (Weeks 5-12)
  └── 100 Medium problems focusing on patterns

Phase 3: Advanced (Weeks 13-20)
  └── 50 Hard problems and company-specific sets

Phase 4: Interview Simulation (Ongoing)
  └── Timed practice and mock interviews
```

## Key Topics to Master

1. **Arrays & Hashing** - Foundation of most problems
2. **Two Pointers & Sliding Window** - Efficient array traversal
3. **Stack & Queue** - Parenthesis and monotonic patterns
4. **Binary Search** - Search optimization technique
5. **Trees & Graphs** - Hierarchical and network data
6. **Dynamic Programming** - Optimization problems
7. **Greedy** - Local optimal choices
8. **Backtracking** - Combinatorial exploration
9. **Heap / Priority Queue** - Order maintenance
10. **Trie & Union-Find** - Advanced data structures

## Quick Start

1. Create an account at [leetcode.com](https://leetcode.com)
2. Start with the "Top 150 Interview Questions" study plan
3. Solve 2-3 problems daily, focusing on understanding patterns
4. Review solutions and discuss with the community
5. Track your progress and revisit weak areas
