# Computer Vision

## Overview

This module covers computer vision fundamentals, techniques, applications, and modern deep learning approaches for visual understanding.

## Topics

- **cv-guide.md** - Complete computer vision fundamentals
- **cv-techniques.md** - Image processing and analysis techniques
- **cv-applications.md** - Real-world computer vision applications
- **cv-interview-questions.md** - Common computer vision interview questions

## Key Areas

| Area | Description |
|------|-------------|
| Image Processing | Filtering, enhancement, transforms |
| Feature Detection | Edges, corners, SIFT, ORB |
| Object Detection | YOLO, R-CNN, SSD |
| Image Segmentation | Semantic, instance, panoptic |
| Image Classification | CNNs, Vision Transformers |
| 3D Vision | Depth estimation, point clouds |

## Prerequisites

- Python and NumPy
- Linear algebra
- Basic deep learning
- Image processing basics

## Getting Started

```bash
pip install opencv-python pillow torchvision matplotlib
```
