# DSA Flashcards

Interactive Q&A flashcards for mastering data structures, algorithms, system design, and networking concepts.

---

## How to Use These Flashcards

### Study Methods

1. **Active Recall** - Try to answer before looking
2. **Spaced Repetition** - Review at increasing intervals
3. **Teach Back** - Explain concepts to someone else
4. **Code It** - Implement the concept after reviewing

### Recommended Study Schedule

| Day | Focus | Time |
|-----|-------|------|
| Day 1 | Data Structures | 30 min |
| Day 2 | Algorithms | 30 min |
| Day 3 | System Design | 30 min |
| Day 4 | Networking | 30 min |
| Day 5 | Review weak areas | 30 min |
| Day 6 | Mixed practice | 30 min |
| Day 7 | Rest / light review | 15 min |

## Learning Tips

### Before Studying
- Review the topic briefly
- Set a specific goal for the session
- Find a quiet study space

### While Studying
- Cover the answer first
- Say the answer out loud
- If wrong, read the explanation
- Mark cards for review

### After Studying
- Review marked cards
- Practice coding the concepts
- Explain to someone else
- Take notes on difficult concepts

## Creating Your Own Cards

### Template

```markdown
## [Concept Name]

**Q:** [Clear, specific question]

**A:** [Concise answer with key points]

**Example:**
```
[Code or diagram example]
```

**Key Points:**
- Point 1
- Point 2
- Point 3

**Common Mistakes:**
- Mistake 1
- Mistake 2
```

### Tips for Good Cards

1. **One concept per card** - Don't combine topics
2. **Clear questions** - Avoid ambiguity
3. **Concise answers** - Key points only
4. **Include examples** - Code or diagrams
5. **Note common mistakes** - Learn from others

## Interview Preparation

### High-Priority Cards

Focus on these before interviews:

1. **Data Structures**

   - Graph algorithms

2. **Algorithms**
   - Sorting algorithms

   - BFS/DFS

3. **Complexity Analysis**
   - Big O notation
   - Space vs Time trade-offs
   - Amortized analysis

4. **System Design**
   - Scalability concepts
   - Caching strategies
   - Database design
