# HackerRank Domains

## Overview

HackerRank organizes problems into domains representing different skill areas. Each domain contains problems of increasing difficulty, allowing you to build expertise progressively.

## Difficulty Progression

### By Domain

**Algorithms**
```
Easy: Warmup + Implementation basics
Medium: Sorting, Search, basic DP
Hard: Graph Theory, advanced DP, Game Theory
```

**Data Structures**
```
Easy: Arrays, basic Linked Lists
Medium: Trees, Stacks, Queues, Heaps
Hard: Tries, Segment Trees, Advanced
```

**Mathematics**
```
Easy: Fundamentals
Medium: Number Theory, Combinatorics
Hard: Geometry, Advanced Algebra
```

## Recommended Practice Order

### Beginner (Weeks 1-4)
1. Algorithms: Warmup, Implementation
2. Data Structures: Arrays
3. Mathematics: Fundamentals

### Intermediate (Weeks 5-12)
1. Algorithms: Sorting, Search, Strings
2. Data Structures: Linked Lists, Trees, Stacks
3. Mathematics: Number Theory

### Advanced (Weeks 13-24)
1. Algorithms: Graph Theory, DP, Bit Manipulation
2. Data Structures: Heaps, Tries, Advanced
3. Mathematics: Combinatorics, Geometry

### Expert (Week 25+)
1. Advanced algorithms and data structures
2. System design concepts
3. Complex problem solving
