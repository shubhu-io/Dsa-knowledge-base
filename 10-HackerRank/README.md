# 10 - HackerRank

## Overview

HackerRank is a leading platform for coding practice, technical interviews, and skill assessment. It is widely used by companies for screening candidates and offers structured learning paths across multiple domains.

## Why HackerRank?

- **Interview Preparation**: Many companies use HackerRank for coding assessments
- **Structured Learning**: Domain-based practice with progressive difficulty
- **Skill Certification**: Earn certificates for demonstrated proficiency
- **Company Challenges**: Participate in sponsored contests
- **Large Problem Archive**: 2,000+ problems across multiple domains

## Quick Start Guide

1. Register at [hackerrank.com](https://www.hackerrank.com)
2. Choose a domain to practice
3. Start with Easy problems
4. Work through the domain sequentially
5. Earn badges and certifications

## Skill Assessment

### Badges
- Earned by solving problems in each domain
- Bronze, Silver, Gold, and Platinum levels
- Track your progress visually

### Certifications
- Verified skill assessments
- Valid for 1 year
- Recognized by employers
- Available in multiple domains

## Company Usage

Many top companies use HackerRank for:
- **Pre-interview screening**: Coding challenges
- **Technical interviews**: Live coding sessions
- **Hiring contests**: Competitive hiring events
- **Skill verification**: Certification validation
