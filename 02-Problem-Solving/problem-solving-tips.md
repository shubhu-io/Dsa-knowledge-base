# Problem Solving Tips

## Top 20 Problem Solving Tips

### Before Coding
1. **Read the problem twice** - Understanding is 80% of the solution
2. **Write down examples** - Visualize input/output
3. **Clarify constraints** - They determine your approach
4. **Ask questions** - Don't assume edge cases

### Design Phase
5. **Start with brute force** - Know the baseline approach
6. **Identify the pattern** - Match with known patterns
7. **Consider multiple approaches** - Compare time/space tradeoffs
8. **Draw diagrams** - Visualize the algorithm

### Implementation
9. **Use meaningful names** - `max_sum` not `ms`
10. **Write modular code** - Break into helper functions
11. **Handle edge cases first** - Empty, null, single element
12. **Use built-in functions** - Don't reinvent the wheel

### Testing
13. **Test with examples** - Verify against given test cases
14. **Test edge cases** - Empty input, maximum constraints
15. **Trace through code** - Walk through with small input
16. **Check for off-by-one errors** - Common in loops/indices

### Optimization
17. **Analyze complexity** - Know your Big O
18. **Look for redundant work** - Cache/memoize if needed
19. **Trade space for time** - Or vice versa
20. **Know when to stop** - Optimal solution exists

## Common Mistakes to Avoid

- Not handling empty input
- Integer overflow
- Infinite loops
- Modifying input unexpectedly
- Forgetting to return value
- Off-by-one errors
- Not considering all edge cases

## Time Management for Interviews

| Phase | Time | Activity |
|-------|------|----------|
| Understanding | 2-3 min | Read, clarify, examples |
| Design | 5-7 min | Approach, complexity analysis |
| Coding | 15-20 min | Implementation |
| Testing | 3-5 min | Verify, edge cases |
| Optimization | 2-3 min | If time permits |

## Practice Strategy

1. **Daily practice** - Solve 1-2 problems daily
2. **Topic-wise** - Master one pattern at a time
3. **Timed practice** - Simulate interview conditions
4. **Review solutions** - Learn from others
5. **Revisit old problems** - Spaced repetition
