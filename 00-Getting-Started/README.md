# Getting Started with the CS & Software Engineering Knowledge Base

Welcome! This repository is a **55-topic reference covering computer science fundamentals, software engineering practices, and interview preparation**. Everything is organized as documentation with code examples — no builds, no tests, no CI/CD.

## How to Use

### 1. Find Your Level

| Level | Start Here | Then Go To |
|-------|-----------|------------|
| **Complete Beginner** | `00-Getting-Started` | `01-Programming-Languages` → `04-Data-Structures` |
| **CS Student** | `04-Data-Structures` | `05-Algorithms` → `11-System-Design` |
| **Job Seeker** | `02-Problem-Solving` | `03-Time-Complexity` → `48-Resume` |
| **Experienced Dev** | `11-System-Design` | `30-DevOps` → `27-AWS` |

### 2. Follow a Learning Path

Each folder has a `README.md` explaining the topic, a main guide, code examples,
and practice problems. Work through them sequentially within each folder.

### 3. Cross-Reference Topics

Use `[[Topic-Name]]` links throughout the documentation to jump between related topics.
The wiki at the GitHub repository mirrors this content with clickable links.

## Study Tips

- **Code everything**: Don't just read — type out and run every example.
- **30 minutes daily** beats 5-hour weekend marathons.
- **Teach what you learn**: Explaining a concept is the best way to solidify it.
- **Build projects**: Apply concepts to real projects, not just isolated exercises.
- **Track progress**: Check off topics as you complete them.
