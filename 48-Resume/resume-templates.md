## Template Selection Guide

### Choose Based on Experience

```
Entry Level (0-2 years)     → Entry-Level Template
Mid Level (3-5 years)       → Software Engineer Template
Senior Level (6+ years)     → Senior Developer Template
Career Changer              → Career Changer Template
Specialized Role             → Role-Specific Template
```

### Choose Based on Style

```
Clean & Professional        → Minimalist Template
Detailed & Comprehensive    → Senior Developer Template
Modern & Creative           → Frontend Developer Template
Technical & Structured      → Backend Developer Template
```

## Template Usage Tips

### Customization Checklist

```markdown
**For Each Application**:
- [ ] Update professional summary for role
- [ ] Adjust skills section to match requirements
- [ ] Tailor bullet points to job description
- [ ] Include relevant keywords
- [ ] Adjust project selection

**Formatting**:
- [ ] Consistent fonts and spacing
- [ ] Proper alignment
- [ ] Appropriate margins
- [ ] Professional file name
- [ ] PDF format
```

### ATS Optimization

```markdown
**Keywords to Include**:
- Match job description terms exactly
- Include technical skills listed
- Use industry-standard terminology
- Include soft skills mentioned

**Format for ATS**:
- Standard section headings
- No tables or columns
- Simple formatting
- PDF or DOCX format
```
