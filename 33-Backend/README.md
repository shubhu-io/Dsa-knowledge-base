## Core Technologies

```
Backend Stack
├── Languages
│   ├── Node.js (JavaScript/TypeScript)
│   ├── Python
│   ├── Java
│   ├── Go
│   └── Rust
│
├── Frameworks
│   ├── Express / Fastify (Node.js)
│   ├── Django / Flask (Python)
│   ├── Spring Boot (Java)
│   ├── Gin / Echo (Go)
│   └── Actix Web (Rust)
│
├── Databases
│   ├── PostgreSQL
│   ├── MySQL
│   ├── MongoDB
│   └── Redis
│
├── Message Queues
│   ├── RabbitMQ
│   ├── Apache Kafka
│   └── Amazon SQS
│
└── Tools
    ├── Docker
    ├── Kubernetes
    ├── Nginx
    └── Prometheus
```

## Key Concepts

### RESTful API Design
- Resource-based URLs
- HTTP methods semantics
- Status codes
- Versioning
- Pagination

### Authentication & Authorization
- JWT tokens
- OAuth 2.0
- Session-based auth
- Role-based access control (RBAC)

### Database Design
- Normalization
- Indexing strategies
- Query optimization
- Connection pooling

### Scalability
- Horizontal vs vertical scaling
- Load balancing
- Caching strategies
- Microservices architecture
