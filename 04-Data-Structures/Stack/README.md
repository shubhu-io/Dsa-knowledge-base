# Stack Data Structure

This directory contains learning materials for stacks, a linear data structure that follows the Last-In-First-Out (LIFO) principle.

## Contents

- [stack-tutorial.md](stack-tutorial.md) - Comprehensive tutorial on stacks
- [stack-problems.md](stack-problems.md) - Practice problems and solutions
- [stack-cheatsheet.md](stack-cheatsheet.md) - Quick reference guide
- [overview.md](overview.md) - Overview of stacks and their place in the curriculum

## Topics Covered

- Stack fundamentals and LIFO principle
- Core operations (push, pop, peek)
- Implementation using arrays and linked lists
- Applications (expression evaluation, backtracking, function calls)
- Variations (min stack, max stack)
- Time and space complexity analysis

## Learning Path

1. Start with the tutorial to understand stack concepts and operations
2. Learn about different implementations and their trade-offs
3. Practice with common stack-based problems
4. Review with the cheat sheet for quick reference
