# Data Structures Overview

## What is a Data Structure?
A way of organizing data for efficient use.

## Main Types
- Linear: Arrays, Linked Lists, Stacks, Queues
- Non-linear: Trees, Graphs, Hash Tables

## Key Concepts
- Access time
- Search time
- Insertion time
- Deletion time
- Space complexity

## Choosing the Right Structure
Consider what operations you need most often.

## Common Operations
- Access: Get element by position/key
- Search: Find element
- Insert: Add element
- Delete: Remove element
- Update: Modify element

## When to Use Which
- Arrays: Fast index access
- Linked Lists: Frequent inserts/deletes
- Stacks: LIFO behavior needed
- Queues: FIFO behavior needed
- Hash Tables: Fast key-based lookup
- Trees: Hierarchical data, sorted access
- Graphs: Network relationships
