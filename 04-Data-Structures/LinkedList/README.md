# Linked List Data Structure

This directory contains learning materials for linked lists, a fundamental linear data structure where elements are not stored in contiguous memory locations but are linked using pointers.

## Contents

- [linked-list-tutorial.md](linked-list-tutorial.md) - Comprehensive tutorial on linked lists
- [linked-list-problems.md](linked-list-problems.md) - Practice problems and solutions
- [linked-list-cheatsheet.md](linked-list-cheatsheet.md) - Quick reference guide
- [overview.md](overview.md) - Overview of linked lists and their place in the curriculum

## Topics Covered

- Singly linked lists
- Doubly linked lists
- Circular linked lists
- Operations (insertion, deletion, traversal)
- Time complexity analysis
- Applications and use cases
- Implementation in multiple programming languages

## Learning Path

1. Begin with the tutorial to understand basic concepts and operations
2. Study different types of linked lists and their variations
3. Practice implementing solutions to common problems
4. Use the cheat sheet for quick reference and review
