# 08 - Codeforces

## Overview

Codeforces is one of the most active competitive programming platforms in the world. It hosts regular contests, maintains a comprehensive problem archive, and features a rigorous rating system that tracks your progress from beginner to grandmaster level.

## Why Codeforces?

- **Regular Contests**: 2-3 contests per week with problems of varying difficulty
- **Active Community**: 400,000+ active competitive programmers worldwide
- **Rigorous Rating System**: Elo-based rating that accurately measures skill
- **Quality Problems**: Well-tested problems with editorial solutions
- **Interview Relevance**: Strong performance indicates solid algorithmic skills

## Contest Format

### Divisions
- **Div. 1**: For rating 1600+ (advanced problems)
- **Div. 2**: For rating < 1600 (intermediate problems)
- **Div. 3**: For rating < 1400 (beginner-friendly)
- **Div. 4**: For rating < 1200 (educational)

### Typical Structure
- 5-7 problems per contest
- Duration: 2 hours
- Scoring: Penalty-based (fewer penalties = better rank)
- Pretest failures add penalty minutes

## Quick Start Guide

1. Register at [codeforces.com](https://codeforces.com)
2. Start with Div. 3 or Div. 4 contests
3. Practice on problemset sorted by rating
4. Read editorials after every contest
5. Upsolve problems you couldn't solve during contest
