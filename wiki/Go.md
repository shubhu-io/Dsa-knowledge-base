= Go =

Go (also known as Golang) is a statically typed, compiled programming language designed at Google. It is known for its simplicity, efficiency, and strong support for concurrent programming.

== Overview ==

Go features:
* Simple and clean syntax
* Fast compilation and execution
* Built-in concurrency support (goroutines, channels)
* Garbage-collected memory management
* Strong standard library with built-in HTTP server and testing tools
* Used in cloud services, microservices, and distributed systems
