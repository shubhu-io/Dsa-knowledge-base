= C++ =

C++ is a general-purpose programming language that was developed as an extension of the C language. It supports object-oriented, generic, and functional programming features in addition to low-level memory manipulation.

== Overview ==

C++ features:
* High performance and fine-grained control over system resources
* Object-oriented programming support (classes, inheritance, polymorphism)
* Generic programming via templates and the Standard Template Library (STL)
* Compatible with C code
* Used in systems programming, game development, and performance-critical applications
