= Rust =

Rust is a systems programming language focused on safety, especially safe concurrency and memory safety without using a garbage collector. It is designed for performance and reliability.

== Overview ==

Rust features:
* Memory safety without garbage collection (ownership system)
* Zero-cost abstractions
* Fearless concurrency
* Pattern matching and algebraic data types
* Compile-time error checking prevents runtime crashes
* Used in systems programming, web assembly, and performance-critical applications
