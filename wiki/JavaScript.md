= JavaScript =

JavaScript is a lightweight, interpreted programming language primarily known as the scripting language for web pages. It is also used in server-side development (Node.js), mobile apps, and desktop applications.

== Overview ==

JavaScript features:
* First-class functions and prototype-based object-oriented programming
* Asynchronous programming with callbacks, promises, and async/await
* Runs in browsers, on servers (Node.js), and in mobile/desktop apps
* Dynamically typed with flexible syntax
* Largest ecosystem of open-source packages (npm)
