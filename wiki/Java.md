= Java =

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is one of the most widely used programming languages in enterprise and Android development.

== Overview ==

Java features:
* Write once, run anywhere (JVM)
* Strong object-oriented foundation
* Automatic memory management (garbage collection)
* Rich standard library and extensive ecosystem
* Platform independence through the Java Virtual Machine
