# Assets

This folder contains visual assets for the repository.

## Structure

```
assets/
├── images/          # Screenshots, logos
├── diagrams/        # Architecture diagrams, flowcharts
└── README.md        # This file
```

## Adding Images

1. Place images in `assets/images/`
2. Reference in markdown:
```markdown
![Alt text](assets/images/filename.png)
```

## Supported Formats

- PNG (recommended for diagrams)
- JPG (for photos)
- SVG (for scalable graphics)
- GIF (for animations)

## Diagram Tools

- [Mermaid](https://mermaid.js.org/) - Markdown-based diagrams
- [Draw.io](https://draw.io) - Free diagram tool
- [Excalidraw](https://excalidraw.com) - Hand-drawn style
