# CodeChef Problem Categories

## Overview

CodeChef organizes problems into categories based on the algorithmic techniques and data structures required. Understanding these categories helps you focus your practice on specific areas.

## Problem Difficulty Structure

### Star-Based Organization

| Stars | Difficulty | Description |
|-------|------------|-------------|
| 1 Star | Beginner | Basic programming concepts |
| 2 Star | Easy | Simple algorithms |
| 3 Star | Medium | Intermediate techniques |
| 4 Star | Hard | Advanced algorithms |
| 5 Star | Very Hard | Expert-level problems |
| 6+ Star | Challenge | Research-level problems |

## Core Categories

### Basic Programming
**Difficulty**: 1-2 Stars
**Topics**:
- Input/Output operations
- Conditional statements
- Loops and iterations
- Basic math operations
- Simple array operations

**Example Problems**:
- Sum of two numbers
- Find maximum in array
- Count vowels in string
- Check if number is prime

---

### Implementation
**Difficulty**: 1-3 Stars
**Topics**:
- Following problem specifications exactly
- Basic data manipulation
- Simple logic implementation
- Edge case handling

**Example Problems**:
- Reverse a number
- Convert temperature
- Calculate factorial
- Check palindrome

---

### Math
**Difficulty**: 2-5 Stars
**Topics**:
- Number theory
- Modular arithmetic
- Combinatorics
- Probability
- Geometry
- Matrix operations

**Difficulty Progression**:
- 2 Stars: Basic arithmetic, GCD, LCM
- 3 Stars: Modular exponentiation, primes
- 4 Stars: Combinatorics, inclusion-exclusion
- 5 Stars: Advanced number theory, matrix exponentiation

---

### Sorting
**Difficulty**: 2-4 Stars
**Topics**:
- Bubble sort, selection sort
- Merge sort, quicksort
- Counting sort
- Custom comparators
- Multi-key sorting

**Difficulty Progression**:
- 2 Stars: Basic sorting, built-in sort
- 3 Stars: Custom comparators, stability
- 4 Stars: Advanced sorting applications

---

### Binary Search
**Difficulty**: 2-4 Stars
**Topics**:
- Binary search on sorted arrays
- Binary search on answer
- Ternary search
- Parametric search

**Difficulty Progression**:
- 2 Stars: Basic binary search
- 3 Stars: Binary search on answer
- 4 Stars: Complex binary search applications

---

### Greedy Algorithms
**Difficulty**: 2-5 Stars
**Topics**:
- Activity selection
- Interval scheduling
- Huffman coding
- Kruskal's MST
- Fractional knapsack

**Difficulty Progression**:
- 2 Stars: Simple greedy choices
- 3 Stars: Prove correctness
- 4 Stars: Complex greedy strategies
- 5 Stars: Advanced optimization

---

### Dynamic Programming
**Difficulty**: 3-6 Stars
**Topics**:
- 1D DP
- 2D DP
- Knapsack variants
- DP on trees
- Bitmask DP
- Digit DP
- DP optimizations

**Difficulty Progression**:
- 3 Stars: Basic 1D/2D DP
- 4 Stars: Knapsack, LIS, LCS
- 5 Stars: Advanced DP, optimizations
- 6 Stars: Complex state transitions

---

### Graphs
**Difficulty**: 3-6 Stars
**Topics**:
- BFS/DFS traversal
- Shortest paths (Dijkstra, Bellman-Ford)
- Minimum spanning trees
- Topological sort
- Strongly connected components
- Network flow
- Bipartite matching

**Difficulty Progression**:
- 3 Stars: BFS, DFS, connected components
- 4 Stars: Shortest paths, MST
- 5 Stars: Advanced graph algorithms
- 6 Stars: Flow, matching, complex graphs

---

### Data Structures
**Difficulty**: 3-6 Stars
**Topics**:
- Arrays, stacks, queues
- Linked lists
- Trees (BST, AVL, segment tree)
- Heaps
- Hash tables
- Union-Find
- Fenwick tree (BIT)
- Sparse table

**Difficulty Progression**:
- 3 Stars: Basic structures, stacks, queues
- 4 Stars: Trees, heaps, BST
- 5 Stars: Segment tree, BIT, advanced
- 6 Stars: Complex structures, lazy propagation

---

### Strings
**Difficulty**: 3-6 Stars
**Topics**:
- String manipulation
- Pattern matching (KMP, Z-algorithm)
- String hashing
- Trie
- Suffix arrays
- Suffix automaton
- Palindromes

**Difficulty Progression**:
- 3 Stars: Basic string operations
- 4 Stars: KMP, hashing, Trie
- 5 Stars: Suffix arrays, advanced patterns
- 6 Stars: Suffix automaton, complex problems

---

### Bit Manipulation
**Difficulty**: 3-5 Stars
**Topics**:
- Bit operations
- Bitmask enumeration
- Bit DP
- Speed tricks
- State compression

**Difficulty Progression**:
- 3 Stars: Basic bit operations
- 4 Stars: Bitmask DP, subset enumeration
- 5 Stars: Advanced bit tricks

## Category-Based Practice

### By Goal

**For Interview Preparation**
Focus on:
- Arrays and Hashing (2-3 Stars)
- Two Pointers (2-3 Stars)
- Binary Search (3-4 Stars)
- Trees (3-4 Stars)
- Graphs (3-4 Stars)
- Dynamic Programming (4-5 Stars)

**For Competitive Programming**
Focus on:
- All categories at 3-5 Stars
- Advanced Data Structures (5-6 Stars)
- Complex Graph Algorithms (5-6 Stars)
- Mathematical Proofs (5-6 Stars)

**For Learning**
Progression:
1. Basic Programming (1 Star)
2. Implementation (1-2 Stars)
3. Math Basics (2 Stars)
4. Sorting (2-3 Stars)
5. Binary Search (3 Stars)
6. Greedy (3 Stars)
7. Dynamic Programming (3-4 Stars)
8. Graphs (3-4 Stars)
9. Data Structures (4-5 Stars)
10. Advanced Topics (5+ Stars)

### By Time Available

**1 Month**
- Master 1-2 Star problems
- Basic 3 Star problems
- Focus: Arrays, Math, Basic Greedy

**3 Months**
- Complete 3 Star problems
- Start 4 Star problems
- Add: DP, Graphs, Data Structures

**6 Months**
- Master 4 Star problems
- Start 5 Star problems
- Add: Advanced algorithms

**1 Year+**
- Master all categories
- Focus on 5-6 Star problems
- Advanced techniques and optimizations

## Problem Selection Strategy

### Step 1: Identify Weak Areas
- Track solve rate by category
- Note which topics take longest
- Identify patterns you struggle with

### Step 2: Focused Practice
- Choose 2-3 categories to focus on
- Solve 20-30 problems per category
- Move to next rating level when comfortable

### Step 3: Mixed Practice
- Alternate between categories
- Solve contest-style mixed problems
- Build versatility

### Step 4: Review and Revise
- Revisit difficult problems
- Review editorials
- Update your notes
