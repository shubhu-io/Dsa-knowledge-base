# 09 - CodeChef

## Overview

CodeChef is a competitive programming platform owned by Unacademy, featuring monthly contests, a large problem archive, and a well-structured rating system. It is particularly popular in India and hosts some of the most prestigious competitive programming contests.

## Why CodeChef?

- **Long Challenge**: Unique format allowing 10 days to solve problems
- **Cook-Off**: Weekly 2.5-hour contests
- **Lunchtime**: Monthly educational contests
- **Structured Learning**: Problems organized by difficulty and topic
- **Strong Community**: Active forums and editorial support

## Contest Types

| Contest | Frequency | Duration | Problems |
|---------|-----------|----------|----------|
| Long Challenge | Monthly | 10 days | 8-10 |
| Cook-Off | Weekly | 2.5 hours | 4-5 |
| Lunchtime | Monthly | 3 hours | 5-6 |
| Starters | Bi-weekly | 2 hours | 5-6 |

## Rating Tiers

| Rating Range | Title | Description |
|--------------|-------|-------------|
| 0 - 999 | Beginner | Learning the basics |
| 1000 - 1399 | Pupil | Solid fundamentals |
| 1400 - 1699 | Specialist | Good problem-solving |
| 1700 - 1999 | Expert | Strong algorithmic skills |
| 2000 - 2199 | Candidate Master | Advanced techniques |
| 2200 - 2499 | Master | Expert-level abilities |
| 2500+ | Grandmaster | Elite programmers |

## Quick Start Guide

1. Register at [codechef.com](https://www.codechef.com)
2. Start with Beginner and 1-star problems
3. Participate in Long Challenge for extended practice
4. Try Cook-Off for timed contest experience
5. Read editorials to learn from each contest
