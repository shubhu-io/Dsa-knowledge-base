## LeetCode

### Overview
- **URL:** leetcode.com
- **Focus:** Technical interview preparation
- **Problems:** 2500+ coding problems
- **Languages:** 20+ supported

### Features
- **Problem Categories:** Arrays, Strings, Trees, Graphs, DP, etc.
- **Difficulty Levels:** Easy, Medium, Hard
- **Company Tags:** Problems from real interviews
- **Discuss:** Community solutions
- **Contests:** Weekly and biweekly

### Pricing
| Plan | Cost | Features |
|------|------|----------|
| Free | $0 | 2000+ problems, basic features |
| Premium | $35/mo | All problems, company tags, interviews |

### Best Practices
1. Follow NeetCode 150 or Blind 75
2. Focus on patterns, not individual problems
3. Time yourself (30-45 min per problem)
4. Review solutions even if you solve it

### Sample Problem Categories
```
Easy (200+)
- Two Sum
- Valid Parentheses
- Merge Sorted Arrays
- Maximum Subarray

Medium (800+)
- Validating BST
- Clone Graph
- LRU Cache
- Serialize/Deserialize Tree

Hard (300+)
- Merge K Sorted Lists
- Alien Dictionary
- Word Ladder II
- Minimum Window Substring
```

## Practice Strategy

### Daily Routine

```
MONDAY: Learn new concept (30 min)
        Solve 2 easy problems (30 min)

TUESDAY: Solve 2 medium problems (60 min)

WEDNESDAY: Review yesterday's problems (30 min)
           Solve 1 medium problem (30 min)

THURSDAY: Solve 2 medium problems (60 min)

FRIDAY: Solve 1 hard problem (60 min)

SATURDAY: Participate in contest (2 hours)

SUNDAY: Review contest problems (1 hour)
        Plan next week
```

### Problem Selection Strategy

**Week 1-4:** Focus on easy problems
- Build confidence
- Learn basic patterns
- Speed up problem solving

**Week 5-8:** Transition to medium problems
- Learn advanced patterns
- Increase complexity
- Time yourself

**Week 9-12:** Tackle hard problems
- Deep problem solving
- Learn optimization
- Master advanced concepts

## Quick Start

### Choosing Your Platform

```
IF interview prep:
    LeetCode (Premium if affordable)
    NeetCode (for structure)

ELSE IF competitive programming:
    Codeforces
    AtCoder

ELSE IF learning concepts:
    GeeksforGeeks
    HackerRank

ELSE IF general practice:
    LeetCode (Free tier)
```

```
□ Choose 1-2 platforms
□ Set up profile

□ Follow beginner problems
□ Track progress
□ Join community
□ Set daily goals
```
