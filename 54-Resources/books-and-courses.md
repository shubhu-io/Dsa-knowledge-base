# Books and Courses for DSA

Curated list of textbooks, online courses, and certifications for mastering Data Structures and Algorithms.

---

## Beginner Level

### Books

#### Grokking Algorithms
- **Author:** Aditya Bhargava
- **Difficulty:** ★☆☆☆☆
- **Best for:** Visual learners, absolute beginners
- **Key Topics:**
  - Binary search
  - Big O notation
  - Arrays and linked lists
  - Recursion
  - Graph algorithms
- **Why Read:** Illustrated guide with Python examples, makes complex topics accessible

#### Python Crash Course
- **Author:** Eric Matthes
- **Difficulty:** ★☆☆☆☆
- **Best for:** Python beginners
- **Key Topics:**
  - Python fundamentals
  - Lists and dictionaries
  - Classes and OOP
  - File I/O
- **Why Read:** Excellent foundation before diving into DSA

## Study Plan by Goal

### For Interview Preparation (3 months)

**Month 1:**
- Read "Cracking the Coding Interview" (Chapters 1-8)
- Complete NeetCode Easy problems
- Take "Algorithms Specialization" Course 1

**Month 2:**
- Read "Cracking the Coding Interview" (Chapters 9-15)
- Complete NeetCode Medium problems
- Take "Algorithms Specialization" Course 2

**Month 3:**
- Complete mock interviews
- Review all problems
- Practice system design basics

### For Deep Understanding (6 months)

**Months 1-2:**
- Read "Introduction to Algorithms" (Chapters 1-15)
- Complete "MIT 6.006" lectures
- Solve 100 problems

**Months 3-4:**
- Read "Introduction to Algorithms" (Chapters 16-25)
- Complete "MIT 6.006" problem sets
- Solve 100 more problems

**Months 5-6:**
- Read "Advanced Algorithms" book
- Work on projects
- Solve 50 hard problems

### For Competitive Programming (12 months)

**Quarter 1:**
- Read "Competitive Programming" (Chapters 1-5)
- Solve 200 easy/medium problems
- Participate in 4 contests

**Quarter 2:**
- Read "Competitive Programming" (Chapters 6-10)
- Solve 200 medium/hard problems
- Participate in 4 contests

**Quarter 3:**
- Read "Competitive Programming" (Chapters 11-15)
- Solve 200 hard problems
- Participate in 4 contests

**Quarter 4:**
- Review and optimize
- Focus on weak areas
- Aim for high contest rankings
