# Deep Learning

## Overview

This module covers deep learning fundamentals, architectures, training techniques, and best practices for building neural networks.

## Topics

- **deep-learning-guide.md** - Complete deep learning fundamentals
- **deep-learning-architectures.md** - CNN, RNN, Transformer, and more
- **deep-learning-training.md** - Optimization, regularization, and training
- **deep-learning-interview-questions.md** - Common deep learning interview questions

## Key Areas

| Area | Description |
|------|-------------|
| Neural Networks | Perceptrons, activation functions, backpropagation |
| CNNs | Convolutional networks for image processing |
| RNNs/LSTMs | Sequential data processing |
| Transformers | Attention mechanisms, self-attention |
| Autoencoders | Unsupervised learning, VAEs |
| GANs | Generative adversarial networks |

## Prerequisites

- Linear algebra and calculus
- Python and NumPy
- Basic machine learning
- GPU availability (optional but recommended)

## Getting Started

```bash
pip install torch torchvision tensorflow keras
```
